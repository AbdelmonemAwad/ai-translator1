## üéØ Overview

AI Translator is a comprehensive media translation system designed to automatically translate movies and TV shows from English to Arabic using cutting-edge AI technologies. The system provides a complete web interface for managing and monitoring all aspects of the translation process.

### Key Highlights
- ü§ñ **AI-Powered**: Uses OpenAI Whisper for speech-to-text and Ollama (Llama 3) for translation
- üåê **Web Interface**: Professional Arabic RTL-supported web dashboard
- üìä **Real-time Monitoring**: System resource monitoring and progress tracking
- üîÑ **Automated Processing**: Batch translation with background task management
- üìÅ **Media Management**: Integration with Sonarr and Radarr for media library management

## üîç Troubleshooting

### Common Issues

**Service Won't Start**
